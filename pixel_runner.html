// Pygame Project

Pixel Runner Game

A 2D endless runner game developed using Python and Pygame featuring obstacle spawning, collision detection, jump mechanics, score tracking, and animated gameplay.

// Features

Core gameplay systems.

Player Movement

Smooth jumping mechanics using gravity simulation and keyboard controls.

Enemy System

Randomized enemy spawning including ninjas and ground enemies.

Collision Detection

Real-time collision logic using Pygame rectangle detection.

Score Tracking

Dynamic score system based on player survival time.

Built with Python & Pygame.

The project was developed entirely using Python and Pygame. It includes sprite rendering, obstacle timers, event handling, collision systems, and gravity physics.
